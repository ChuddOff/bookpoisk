# Буквопоиск

Монорепозиторий:

- frontend/ — Web (React + Vite) + Telegram Mini App маршруты
- backend/ — API (заполнит команда бекенда)
- model/ — LM (заполнит команда LM)

## Frontend старт

```bash
cd frontend
npm i
npm run dev
```

## Model старт
```bash
cd model
pip install -r requirements.txt
python3 main.py
```

## Требования

- Node LTS (см. .nvmrc)
- WSL2 для Windows
